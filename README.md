# TallySync

**Professional Desktop Application for TallyPrime Integration**

## Overview

TallySync is a production-grade Windows desktop application that connects with TallyPrime using Tally's built-in XML interface.

## Technology Stack

- Python 3.10+
- CustomTkinter
- SQLite
- XML
- ReportLab

## Installation

1. Clone repository
2. Create virtual environment: `python -m venv venv`
3. Activate: `.\venv\Scripts\Activate.ps1`
4. Install: `pip install -r requirements.txt`
5. Run: `python -m app.main`
